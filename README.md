

# X-Reader 📖

**英文 PDF 智能阅读器** — 专为英语学习者打造

---

## ✨ 功能特色

### 🔄 实时翻译
选中 PDF 中的英文句子，**自动翻译**为中文。基于Apple本地翻译引擎。支持 Google 翻译引擎，无需任何配置，开箱即用。支持百度翻译引擎（需输入ID和KEY）

### 🎧 AI 语音朗读
内置 **Kokoro AI 语音引擎**（纯 CoreML，完全离线），54 种高质量英语声音可选。首次使用自动下载模型（约 99MB），之后全程离线运行，不消耗流量。

### 📝 语法分析
- **词性标注**：每个单词自动标注名词、动词、形容词、副词等
- **单词分级**：标注每个单词的难度等级（A1~C2）

### 📑 阅读辅助
- **自动目录**：提取 PDF 内嵌目录，快速跳转章节
- **书签系统**：为常用页面添加书签，自动记忆阅读位置
- **全文搜索**：快速定位 PDF 中的关键词
- **OCR 识别**：支持扫描版 PDF 文字识别（需 macOS 15+）

### 🎨 个性化
- **深色/浅色/跟随系统** 三种主题
- **中英文界面** 即时切换
- 可调节朗读语速

## 🚀 快速开始

### 系统要求
- macOS 15.0 (Sequoia) 或更高版本
- Apple Silicon (M1/M2/M3/M4) 或 Intel 芯片

### 安装
1. 下载 `X-Reader .app`
2. 拖动到「应用程序」文件夹
3. 首次打开：右键 → 打开（如果遇到 Gatekeeper 提示）

### 使用方法
1. **打开 PDF** → 点击「打开文件」或拖拽 PDF 到窗口
2. **选中文字** → 自动翻译 + 朗读 + 语法分析
3. **调整朗读** → 在右侧面板选择声音和语速
4. **添加书签** → 点击书签按钮或右键菜单

| 快捷键 | 功能 |
|--------|------|
| `⌘O` | 打开 PDF 文件 |
| `⌘N` | 多开 |
| `⌘F` | 搜索 |
| `⌘D` | 添加/移除书签 |
| `⌃⌘F` | 全屏 |
| `←` `→` | 上一页/下一页 |
| `⌘+` `⌘-` | 放大/缩小 |

## 🏗️ 技术栈

- **SwiftUI** + **PDFKit** — 原生 macOS 应用
- **Kokoro CoreML** — 本地 AI 语音引擎
- **Apple NaturalLanguage** — 词性标注
- **Apple Translate API** — 翻译引擎
- **Google Translate API** — 翻译引擎
